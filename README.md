# Проект Django DRF

## Описание
Проект системы управления обучением (LMS) с использованием Django и Django REST Framework.

## Функционал
- Кастомная модель пользователя с email-авторизацией
- Модели курсов и уроков
- Система платежей
- API с CRUD операциями
- Фильтрация и сортировка платежей
